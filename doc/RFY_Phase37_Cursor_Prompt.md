# ROOM FOR YOU — Phase 37 Cursor Prompt
## Public Site Redesign — Crimson Red · Clean Minimal · Human

---

## CONTEXT

Redesign the public site color system and feel. The site currently uses gold (`#C9A84C`) as its accent color and feels AI-generated. Replace gold with deep crimson red and make it feel genuinely human — like it was designed by a person with taste, not generated by a tool.

**Design direction:**
- **Accent color:** Deep crimson red `#8B0000` (primary), `#A00000` (hover/bright variant)
- **Background:** Keep dark `#0F0F0F`
- **Feel:** Clean, minimal, editorial — white space, sharp typography, human
- **What makes it unforgettable:** The restraint. Less is more. Every element earns its place.

**Key principle:** Remove anything that feels "template-ish". No generic gradients, no predictable layouts. Think about how a real gospel ministry would present itself — with dignity, weight, and warmth.


## TASK 12 — Remove AI-Template Patterns

These specific patterns make the site feel AI-generated. Fix them:

1. **Remove** any `bg-gradient-to-r from-gold to-gold-bright` type gradient backgrounds — use flat crimson instead
2. **Simplify** any element that has too many layered effects (multiple box-shadows, multiple gradients, glow + border + shadow all at once)
3. **Reduce** animation complexity — if something has 3 animations, cut to 1. Keep only the most impactful animation per section.
4. **Add breathing room** — increase padding on sections that feel cramped. Sections should have at least `py-24` (96px) padding.
5. **Remove** any `.animate-breathe` or pulsing glow effects that feel like templates

## NOTES FOR CURSOR

- The admin dashboard uses `--a-gold` CSS variables — do NOT touch these. Only change public-facing components.
- Crimson `#8B0000` is a darker, more serious red. It will look powerful on the dark background. Trust it.
- The `.text-gold-gradient` class used gradient text which was flashy. Replace it with flat crimson `color: #8B0000` — simplicity reads as confidence.
- The CTA section background change from gold to crimson is the most impactful single change — the whole page energy shifts.
- Deep crimson on black is a classic combination used in church bulletins, hymn books, and ministerial design for centuries. It carries weight and dignity. Lean into this.
- `#8B0000` at full opacity on `#0F0F0F` has strong contrast. The ratio is high enough for text use as well.
- Do not change the Cloudinary OG image — that's a photograph, not a design element.
- The chat widget (`ChatWidget.tsx`) uses gold `#C9A84C` — update the bubble and send button to crimson `#8B0000` as well.
